Shape（圖形）, Circle（圓）, Rectangle（矩形）, Square（正方形）, Triangle（三角形）。  
需要設計成 Structure/Class，圓有半徑，矩形有寬和高，正方形有邊長，三角形有三個邊。 

輸入說明 ： 

第一行輸入圖形個數 N。 

第二行到第 N+1 行輸入圖形種類、以及該圖形所需整數資料，以空白間隔。 

圖形種類以字元表示，C 代表圓、R 代表矩形、S 代表正方形、T 代表三角形。 

如果輸入 C 要跟隨一個數值為半徑，  
輸入 R 要跟隨兩個數值寬和高，  
輸入 S 要跟隨一個數值代表邊長，  
輸入 T 要跟隨三個數值代表三個邊。 


輸出說明： 

輸出 N+1行， 

前 N 行整數，代表 N 個圖形的周長。 

第 N+1行整數，代表 N 個圖形的周長總和。 

PI 設為 4。 

Sample Input:  
5  
T 3 4 5  
S 1  
R 2 3  
C 1  
T 3 4 6 

Sample Output:  
12  
4  
10  
8  
13  
47
